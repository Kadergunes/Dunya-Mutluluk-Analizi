Project Goal:
This project aims to visualize and analyze the impact of factors such as economic status, freedom, and health on countries' happiness scores using the World Happiness Report data.

Analysis and Visualizations:
Upon examining the relationships between variables, it is observed that the factors most affecting happiness are Economy (0.78), Family (0.74), and Health (0.72).
